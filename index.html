🐍 Python Basics

📦 Variables and Assignment

Variables are like boxes where we store information. Let's learn how to create them!

name = 'Alice'
age = 12
print(f'Hi, I am {name} and I am {age} years old!')
Output: Hi, I am Alice and I am 12 years old!

🖨️ Print Statements

Print statements help us show messages on the screen. It's like talking to the computer!

print('Hello, World!')
print('🐍 Python is fun!')
print('*' * 10)
Output:
Hello, World!
🐍 Python is fun!
**********

💭 Comments and Documentation

Comments are notes we write for ourselves. The computer ignores them!

# This is a comment
print('Hello!')  # This prints a greeting
Output: Hello!

🏷️ Data Types

Python can work with different types of information: text, numbers, and true/false values!

text = 'Hello'  # String
number = 42      # Integer
price = 9.99     # Float
is_fun = True    # Boolean

🤔 Making Decisions

60% Complete

✅ Boolean Logic

Boolean logic helps computers make yes/no decisions, just like we do!

is_sunny = True
is_weekend = False
print(is_sunny and is_weekend)
Output: False

⚖️ Comparison Operators

We can compare things using special symbols like >, <, and ==

age = 13
if age >= 13:
    print('You are a teenager!')
else:
    print('You are a kid!')
Output: You are a teenager!

🌦️ If Statements

If statements help our program make different choices based on conditions!

weather = 'sunny'
if weather == 'rainy':
    print('☔ Take an umbrella!')
elif weather == 'sunny':
    print('😎 Wear sunglasses!')
else:
    print('🧥 Take a jacket!')
Output: 😎 Wear sunglasses!

🧠 Logical Operators

We can combine conditions using 'and', 'or', and 'not' - just like in real life!

homework_done = True
room_clean = False
if homework_done and room_clean:
    print('You can play games!')
else:
    print('Finish your tasks first!')
Output: Finish your tasks first!

🔄 Repeating Actions

40% Complete

🎯 For Loops

For loops help us repeat actions a specific number of times!

for i in range(5):
    print('⭐' * (i + 1))
Output:
⭐
⭐⭐
⭐⭐⭐
⭐⭐⭐⭐
⭐⭐⭐⭐⭐

⏰ While Loops

While loops keep repeating as long as a condition is true!

count = 0
while count < 3:
    print(f'Count: {count}')
    count += 1
Output:
Count: 0
Count: 1
Count: 2

🛑 Break & Continue

Sometimes we want to stop a loop early or skip to the next round!

for num in range(10):
    if num == 5:
        break
    print(num)
Output: 0 1 2 3 4

🔁 Nested Loops

We can put loops inside other loops to create patterns!

for row in range(3):
    for col in range(3):
        print('🟦', end=' ')
    print()
Output:
🟦 🟦 🟦
🟦 🟦 🟦
🟦 🟦 🟦

📦 Storing Data

20% Complete

📝 Lists

Lists help us store multiple items together, like a shopping list!

fruits = ['apple', 'banana', 'orange']
print(f'I have {len(fruits)} fruits')
fruits.append('grape')
print(fruits)
Output:
I have 3 fruits
['apple', 'banana', 'orange', 'grape']

📚 Dictionaries

Dictionaries store information with labels, like a real dictionary!

student = {
    'name': 'Alex',
    'age': 12,
    'grade': 'A'
}
print(f"Student: {student['name']}")
Output: Student: Alex

🔤 String Methods

Strings have special powers! We can make them uppercase, lowercase, and more!

message = 'hello world'
print(message.upper())    # HELLO WORLD
print(message.title())    # Hello World
print(message.count('l'))  # 3
Output:
HELLO WORLD
Hello World
3

📍 Tuples

Tuples are like lists but they can't be changed once created!

coordinates = (10, 20)
x, y = coordinates
print(f'X: {x}, Y: {y}')
Output: X: 10, Y: 20

⚙️ Creating Functions

0% Complete

🛠️ Function Definition

Functions are like recipes - they take ingredients and make something new!

def greet(name):
    return f'Hello, {name}!'

message = greet('Alice')
print(message)
Output: Hello, Alice!

📥 Parameters

Parameters are the ingredients we give to our function recipes!

def add_numbers(a, b):
    result = a + b
    return result

sum_result = add_numbers(5, 3)
print(f'5 + 3 = {sum_result}')
Output: 5 + 3 = 8

📤 Return Values

Functions can give us back a result, like a vending machine!

def calculate_area(length, width):
    area = length * width
    return area

room_area = calculate_area(10, 12)
print(f'Room area: {room_area} sq ft')
Output: Room area: 120 sq ft

🌍 Scope

Variables have homes - some live everywhere, others live only inside functions!

global_var = 'I am global'

def my_function():
    local_var = 'I am local'
    print(global_var)
    print(local_var)

my_function()
Output:
I am global
I am local
